# thack-berlin
THack Berlin 2015

# How to install

* Install [vagrant](https://www.vagrantup.com/) and NFS server

* `git clone [this repo]`

* `cd your dir`

* `vagrant up`

* `vagrant ssh`

* `cd /vagrant`

* `composer install`

# Run

Visit http://thack-berlin.vagrant.netanday.it/app_dev.php/home
